# Security policy

## Reporting a vulnerability

If you find a security issue in this project, please **do not** open a public
GitHub issue. Instead, open a private security advisory on this repository
or contact the maintainer directly.

This project is a community / personal project and is **not an official
Microsoft product**. Vulnerabilities in the underlying Azure Container Apps
Sandboxes service or the Azure SDKs should be reported through Microsoft's
official channels (https://msrc.microsoft.com/), not here.

## Supported versions

Only the latest minor release receives security fixes.
